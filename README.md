# Setup for Firefox
### 1. Open Firefox and Hit [ Control + B ] Open a Bookmark.
### 2. Select AnyBook For Exmaple (i am selecting Kali Docs.)

<img width="1199" height="512" alt="image" src="https://github.com/0xmrsecurity/Scrad/blob/main/Img/step2.png" />

### 3. Right click it and Edit Bookmark ...

<img width="1199" height="512" alt="image" src="https://github.com/0xmrsecurity/Scrad/blob/main/Img/step3.png" />

### 4. Paste the Scrad Code in the Bookmark url and Rename the Bookmark as Scrad.

<img width="1199" height="512" alt="image" src="https://github.com/0xmrsecurity/Scrad/blob/main/Img/step4.png" />

### 5. Finally save it. Open any Demo Website Example 0xmrsecurity.github.io/ and know click on the save Bookmark called  scrad.
